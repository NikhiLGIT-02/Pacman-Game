# Pac-Man AI

A modern implementation of the classic **Pac-Man** game developed in **Python** using **Pygame**. The project demonstrates core Artificial Intelligence concepts through intelligent ghost behavior, state-based decision making, and heuristic path selection while delivering a smooth and interactive gameplay experience.

---

## Features

- Classic Pac-Man gameplay
- Intelligent ghost AI with unique behaviors
- Multiple ghost modes (Chase, Scatter, Frightened, Eaten)
- Multiple difficulty levels
- Score and high-score tracking
- Collision detection
- Sound effects and particle animations
- Level progression
- Modern user interface

---

## Technologies Used

- Python 3
- Pygame

---

## AI Concepts

- Intelligent Agents
- Rule-Based AI

## Installation

Clone the repository:

```bash
git clone https://github.com/NikhilGIT-02/Pac-Man-AI.git
```

Navigate to the project directory:

```bash
cd Pac-Man-AI
```

Install the required package:

```bash
pip install pygame
```

Run the application:

```bash
python main.py
```

---

## How to Play

- Use the **Arrow Keys** or **WASD** to move Pac-Man.
- Eat all pellets to complete the level.
- Avoid ghosts unless they are in **Frightened Mode** after consuming a power pellet.
- Clear all pellets to advance to the next level.
